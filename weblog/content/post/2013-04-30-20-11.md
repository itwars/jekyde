---
title: Configuration
name: configuration
---

You may change configurations by editing `config.yml` file in your template folder. The following file shows the default settings:

```yaml
## Website
title:              Simple Blog
subtitle:           A static blog generator

## Output
baseurl:            /
archive_dir:        archive
post_link:          post/:year/:month/:day/:name.html
page_link:          page/:name
paginate:           4

## Engine
latex:              false

## Server
port:               4040
```
